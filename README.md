# song-vue-template

vue 组件

## 安装

```
yarn add -D song-vue-template
```

### 在 vue 中使用组件

1. 在 main.js 文件中写入

```javascript
import Svt from 'song-vue-template'

import 'song-vue-template/dist/svt.css'

Vue.use(Svt)
```

2. 在开发的 vue 页面中使用

```
<svt-template/>
```
